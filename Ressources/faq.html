1 Qu’est ce que les fichiers .dic

Les fichiers .dic sont des fichiers texte. On aurait pu utiliser l’extension .txt. L’idée de l’extension .dic est de bien les identifier comme un dictionnaire.

La règle pour ces fichiers est qu’il n’y a qu’un seul mot par ligne.

Ce fichier se lit donc comme un fichier texte classique (open en Python). Pour chaque ligne, on pourra utiliser la méthode strip() de str pour supprimer les blancs en début et fin de ligne.

2 L’algorithme donné pour la distance

2.1 Erreur dans l’énoncé.

La ligne

Si i > 1 Et j > 1 Et a[i] = b[j-1] Et a[i-1] = b[j] Alors

doit être remplacée par :

Si i > 1 Et j > 1 Et a[i-1] = b[j-2] Et a[i-2] = b[j-1] Alors

2.2 Mauvais résultat

La distance trouvée pour les mots ‘ma’ et ‘amx’ est 3 au lieu de 2.

On pourrait en effet faire ‘ma’ -> ‘am’ -> ‘amx’.

C’est parce que l’algorihtme donné ne traite pas ce cas où une même sous-chaîne subie plusieurs transformations. Ici, il fait deux modification sur la même sous-chaîne ‘ma’ : une transposition puis un ajout.

Plus d’informations sur Wikipedia.

Nous limiterons à l’algorithme qui est donné dans le sujet.

5 J’ai des problèmes quand je lis en Python le dictionnaire (caractères bizarres)

Les fichiers fournis sont un utf-8. Si la lecture en Python affiche des caractères bizarres, vous pouvez utiliser le paramètre optionnel encoding de open :

with open("monfichier.txt", "r", encoding='utf-8') as fichier:
    ...

6 Comment faire pour écrire dans un fichier texte en Python ?

On peut utiliser print en fournissant le paramètre optionnel file :

with open("monfichier.txt", "w") as fichier:
    print("Ce que je veux....", file=fichier)

7 Comment identifier les mots après avoir fait re.split ?

L’appel à re.split indiqué dans le sujet garantit que les mots sont toujours à indice pair. On peut donc s’appuyer sur cette propriété. Sinon, on peut tester un objet de type str avec la méthode isalnum pour vérifier que tous ses caractères sont des lettres ou des chiffres.

>>> 'çà'.isalnum()
True
>>> ', '.isalnum()
False
>>> 'çà1'.isalnum()
True

8 Comment choisit-on le mot pour corriger dans la version automatique ?

À vous de choisir. Il faut juste que ce soit l’un des mots qui a la plus petite distance au mot à corriger.

9 Comment faire pour trouver les mots à la distance de N dans la stratégie 2 ?

On peut raisonner par récurrence. Les mots à la distance 0, c’est le mot lui-même.

À la distance 1, on applique toute les transformations élémentaires 1 fois au mot initial.

À la distance N+1, on suppose que l’on connait les mots à la distance N. Les mots à la distance N+1 sont les mots obtenus à partir de ceux à une distance N en appliquant une transformation élémentaire.

Attention : pour le calcul de la distance, on peut retomber sur un mot déjà vu à une distance inférieure.

12 Comment connaître l’alphabet ?

Définir explicitement une liste avec les caractères possibles d’un mot est dangereux (risque d’en oublier) et n’est pas général (une autre langue peut avoir un alphabet différent).

Une meilleur solution est de considérer que l’alphabet est défini par les mots du dictionnaire. À ne calculer qu’une seule fois, bien sûr !

L’avantage supplémentaire, c’est que la stratégie 2 sera plus facile à tester si l’alphabet est petit, donc si vous avez un dictionnaire dont les mots utilisent peu de lettres.

13 Comment écrire un fichier texte à partir d’une liste de mots ?

Trois choses à utiliser :

  • open avec 'w' comme deuxième paramètre pour ouvrir le fichier en écriture
  • print avec l’option file pour préciser le fichier dans lequel print va écrire et end pour éviter d’avoir des retours à la ligne non souhaités
  • join de str pour concaténer les mots d’une liste (d’un itérable) en ajoutant un séparateur entre eux

Exemple :

mots = ['un', ' ', 'exemple', ' !', '\n', 'sur', ' ', 'deux', '   ', 'lignes']
with open ('resultat-tmp.txt', 'w') as f:
    print(''.join(mots), file=f, end='')
